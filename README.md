# Custom PC Build & Windows 7 ‚Üí Windows 10 Pro Upgrade

This repository documents a custom-built desktop PC assembled using a mix of **new, enthusiast-grade components** and **salvaged hardware**, originally configured with **Windows 7** and later upgraded in-place to **Windows 10 Pro**.

A key goal of this project was maximizing reuse of existing hardware while maintaining long-term system stability and performance.

This was my **first PC build**, and it marked the beginning of my journey into IT and hands-on systems work.

## üß© Project Background

This system was assembled by:
1. Reusing a functional laptop SSD as the system disk
2. Installing new desktop-class components around that storage
3. Housing the system in a spare CyberPowerPC case
4. Installing and wiring the power supply manually
5. Installing and configuring case fans for airflow
6. Running Windows 7 initially for compatibility
7. Performing an in-place upgrade to Windows 10 Pro without data loss

Despite the unconventional mix of components, the system has remained stable and fully functional.
